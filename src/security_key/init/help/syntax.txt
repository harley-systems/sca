@@@OPTIONS@@@

The entity parameter is the type of the entity for which we want to initialize
the security key for. The sca configuration file assigns a cryptographic token
storage device (a security key) to each of the declared default entity types
(ca, subca, user, service, host). The related setting name is:
<entity>_security_key_id.
