sca [<sca_options>] display [<display_options>] <document_type> <entity> [<entity_id>]

@@@SCA OPTIONS@@@

@@@OPTIONS@@@

allowed values for the document_type are:

  - csr           - display certificate signing request
  - crt           - display certificate
  - key           - display private key
  - pub           - display public key
  - pub_ssh       - display

allowed values for the entity are:

  - ca            - for current certificate authority
  - subca         - for current sub-ordinate certificate authority
  - user          - for current user
  - host          - for current host
  - service       - for current service

entity_id argument is optional and it stands for the id of the given entity.
when omitted, the current entity id determined by the environment variables is
user.
